    Utility Functions

    Utility functions

    LazySets.unit_step — Method.
    unit_step(x)

    The unit step function, which returns 1 if and only if x is greater or equal than zero.

    Input

    • x – a floating point number

    Notes

    This function can be used with vector-valued arguments via the dot operator.

    Examples

    julia> unit_step([-0.6, 1.3, 0.0])
3-element Array{Float64,1}:
 -1.0
 1.0
 1.0

    LazySets.jump2pi — Function.
    jump2pi(x)

    Return $x + 2π$ and only if $x$ is negative.

    Input

    • x – a floating point number

    Examples

    julia> jump2pi(0.0)
0.0
julia> jump2pi(-0.5)
5.783185307179586
julia> jump2pi(0.5)
0.5

    Base.:<= — Method.
    u <= v

    States if arg(u) [2π] <= arg(v) [2π].

    Input

    • u – a first direction

    • v – a second direction

    Output

    True iff arg(u) [2π] <= arg(v) [2π]

    Notes

    The argument is measured in counter-clockwise fashion, with the 0 being the direction (1, 0).
